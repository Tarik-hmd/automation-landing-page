# EasyWay – Automation & AI Services Landing Page ⚡

This project is a responsive landing page built for an **automation & AI services business**.  
It showcases how businesses can save time by automating workflows, bookings, and client communication.

The goal of this project is to demonstrate:
- clean frontend structure (HTML & CSS)
- clear service positioning
- real-world automation use cases
- portfolio-ready presentation

---

##  Features

- Modern, clean SaaS-style design
- Sticky and minimal navigation bar
- Services section focused on automation & AI
- Portfolio section with realistic automation examples
- Call-to-action with scheduling link
- Fully responsive layout

---

##  Automation Use Cases Shown

- AI Booking Assistant (scheduling, reminders, rescheduling)
- Lead Capture & CRM Automation
- Invoice & Payment Automation
- Tool integrations (Google, WhatsApp, CRM, Stripe)

These examples represent real automation workflows used by modern businesses.

---

##  Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- No frameworks (pure frontend for learning & clarity)
